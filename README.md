# Briefed - Client Brief Tool for Creatives

## The Problem
Creative freelancers receive briefs via 45 voice notes, scattered emails, and blurry screenshots. This leads to scope creep, miscommunication, and wasted time.

## The Solution
Brief Builder provides a guided form for clients to fill out, generating a structured PDF brief that freelancers can reference throughout the project.

## Tech Stack
- Next.js, TypeScript, Tailwind
- Prisma + PostgreSQL
- Clerk Auth
- Uploadthing for file uploads
- react-pdf for PDF generation
